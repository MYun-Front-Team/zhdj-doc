# 新增部门

_**【接口地址】**_

http://ip:port/FJAcion/Adaption/AddDepartment

> #### _请求数据_

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| Name | string | 是 | 部门名称 |
| Master | string | 是 | 负责人 |
| Phone | string | 是 | 联系电话 |
| AreaCode | string | 是 | 区域编号 |

> #### _应答数据_

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| Body | int | 是 | 部门id |
