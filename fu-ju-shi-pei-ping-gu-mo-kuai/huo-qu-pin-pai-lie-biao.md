# 获取品牌列表

_**【接口地址】**_

http://ip:port/FJQuery/Adaption/GetBrandList

> #### _请求数据_

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| Name | string | 否 | 品牌名称 |

> #### _应答数据_

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| BrandSysNo | int | 是 | 品牌id |
| Name | string | 是 | 品牌名称 |
| Master | string | 是 | 负责人 |
| Phone | string | 是 | 联系电话 |
| IsEnable | int | 是 | 1启用，0禁用 |
