# 获取供应商列表

_**【接口地址】**_

http://ip:port/FJQuery/Adaption/GetSupplierList

> #### _请求数据_

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| Name | string | 否 | 供应商名称 |

> #### _应答数据_

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| SupplierSysNo | int | 是 | 供应商id |
| Name | string | 是 | 供应商名称 |
| Master | string | 是 | 联系人 |
| Phone | string | 是 | 联系电话 |
| Email | string | 是 | 联系邮箱 |
| IsEnable | int | 是 | 1启用，0禁用 |
