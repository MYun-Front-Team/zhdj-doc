# 获取待评估数量

_**【接口地址】**_

http://ip:port/FJQuery/Adaption/GetWaitAssessNum

> #### _请求数据_

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| PersonSysNo | int | 是 | 服务人员id |

> #### _应答数据_

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| WaitAssessNum | int | 是 | 待评估数量 |
| NewNum | int | 是 | 新增评估数量 |
