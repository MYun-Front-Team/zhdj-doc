# 获取生活辅具列表

_**【接口地址】**_

http://ip:port/FJQuery/Adaption/GetOtherAssistList

> #### _请求数据_

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| DisabilityTypeList | array int | 是 | 残障类别列表 |

> #### _应答数据_

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| OtherAssistSysNo | int | 是 | 生活辅具id |
| DisabilityType | int | 是 | 残障类别 |
| OtherAssistName | string | 是 | 生活辅具名称 |
