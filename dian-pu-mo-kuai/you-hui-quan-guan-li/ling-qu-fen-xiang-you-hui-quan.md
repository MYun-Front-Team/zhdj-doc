# 领取分享优惠券

##### _【功能说明】_ {#【功能说明】}

领取优惠券

_**【应用场景】**_

领取优惠券

注：如果未提供Person参数，那么使用操作人。

_**【接口地址】**_

[http://ip:port/ShopAction/Coupon/PickShareCoupon](http://ip:port/ShopAction/Coupon/PickShareCoupon)

> #### _请求数据_ {#请求数据}

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| CouponCodeSysNo | int | 是 | 领取的优惠券码系统编码 |
| PersonSysNo | int | 否 | 人员系统编码 |
| CouponCode | string | 否 | 优惠券编码 |
