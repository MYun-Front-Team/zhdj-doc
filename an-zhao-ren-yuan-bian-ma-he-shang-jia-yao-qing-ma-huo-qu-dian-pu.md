# 按照人员编码和商家邀请码获取店铺

##### _【功能说明】_ {#【功能说明】}

按照人员编码和商家邀请码获取店铺实体


_**【应用场景】**_

按照人员编码和商家邀请码获取店铺实体

_**【接口地址】**_

http://ip:port/ShopQuery/Shop/GetShopBySellerCode

> #### _请求数据_ {#请求数据}

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| SellerCode | string | 是 | 商家编码 |
| PersonSysNo | int | 是 |人员编码 | 

> #### _应答数据 _ {#应答数据-（巡河记录数组）}

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| ShopBase | object | 是 | 基础字段 |
| ShopStatistic | object | 否 | 统计字段 |
