# 查询三方会员卡余额

##### _【功能说明】_ {#【功能说明】}

查询三方会员卡余额

_**【应用场景】**_

查询三方会员卡余额

_**【接口地址】**_

http://ip:port/WalletQuery/MemberCard/GetAccDBMoney

> #### _请求数据_ {#请求数据}

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| CardNo | string | 否 | 三方卡号 |
| EDIName| string | 否 |三方机构编码（目前写死 zytk_getaccdbmoney） |


> #### _应答数据 _ {#应答数据-（巡河记录数组）}

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| Balance| decimal | 是 | 余额 |
