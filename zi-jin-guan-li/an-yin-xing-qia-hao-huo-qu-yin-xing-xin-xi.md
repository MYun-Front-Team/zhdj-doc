# 按银行卡号获取银行信息

##### _【功能说明】_ {#【功能说明】}

按银行卡号获取银行信息

_**【应用场景】**_

按银行卡号获取银行信息

_**【接口地址】**_

http://ip:port/WalletQuery/Bank/GetBankInofByCode

> #### _请求数据_ {#请求数据}

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| CardNo | string | 是 | 银行卡号 |


> #### _应答数据 （数组）_ {#应答数据-（巡河记录数组）}

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| BankSysNo | int | 是 | 银行系统编码 |
| BankName | string | 是 | 银行名称 |
| BankUrl | string | 否 | 银行logo |
