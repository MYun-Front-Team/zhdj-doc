# 新增路线 {#获取河长巡河记录}

_**【接口地址】**_

http://ip:port/EqmAction/Equipment/AddLine

> #### _请求数据_ {#请求数据}

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| LineName | string | 是 | 线路名称 |
| PersonSysNo | int | 是 | 线路所属运营商人员编码 |
|  |  |  |  |

> #### _应答数据 _ {#应答数据-（巡河记录数组）}

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| Body | int | 是 | 线路编码 |
