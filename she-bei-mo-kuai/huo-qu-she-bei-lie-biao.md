[http://ip:port/EqmQuery/Equipment/GetEquipmentList](http://ip:port/EqmQuery/Equipment/GetEquipmentList)

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| Query | object | 是 | 见搜索条件 |

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| Body | array Equipment | 是 | 数据列表 |
