# 确认迁移商家

##### _【功能说明】_ {#【功能说明】}

确认迁移商家

_**【应用场景】**_

确认迁移商家



_**【接口地址】**_

http://ip:port/OrganizationAction/Seller/ConfirmTransfer


> #### _请求数据_ {#请求数据}

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| TransferSysNo | int | 是 | 迁移系统编码 |
| TransferStatus | int | 是 | 迁移状态（10已完成，11拒绝）|

#### _应答数据 _ {#应答数据-}

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
|  | int | 是 | 迁移后给老商家分配的新组织号（原组织号迁移到新店长那里了） |
