# 设置组织人员为部门负责人
##### _【功能说明】_ {#【功能说明】}

设置组织人员为部门负责人

_**【应用场景】**_

设置组织人员为部门负责人

_**【接口地址】**_

http://ip:port/OrganizationAction/Organization/SetOrganizationPersonBoss

> #### _请求数据_ {#请求数据}

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| OrganizationSysNo | int | 否 | 组织系统编码 |
| PersonSysNo | int | 否 | 人员系统编码 |
| IsDepartmentBoss | int | 否 | 是否部门领导|
