# 设置微信OPENID的上级邀请码

##### _【功能说明】_ {#【功能说明】}

设置微信OPENID的上级邀请码

_**【应用场景】**_

设置微信OPENID的上级邀请码

_**【接口地址】**_

http://ip:port/BasicAction/Basic/SetHigherInvitation


> #### _请求数据_ {#请求数据}

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| WxOpenID| string | 是 | WxOpenID |
| InvitationCode| string | 是 | 上级邀请码 |
