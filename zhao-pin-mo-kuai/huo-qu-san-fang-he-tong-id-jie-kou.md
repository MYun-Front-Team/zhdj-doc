# 获取三方合同ID接口
##### _【功能说明】_ {#【功能说明】}
获取三方合同ID接口


_**【应用场景】**_

获取三方合同ID接口


_**【接口地址】**_

http://ip:port/RecruitAction/Contract/SetContractThreePartyContractId

> #### _请求数据_ {#请求数据}

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| ContractSysNo| int | 是 | 合同系统编码 |
| PersonSysNo| int | 是 | 人员系统编码 |

> #### 返回数据

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| | int | 是 | 三方合同编码 |
