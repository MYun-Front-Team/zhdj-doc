# 解绑银行卡或支付宝

##### _【功能说明】_ {#【功能说明】}

解绑银行卡或支付宝

_**【应用场景】**_
解绑银行卡或支付宝



_**【接口地址】**_

http://ip:port/RecruitAction/MyBank/UnBindCard

> #### _请求数据_ {#请求数据}

| 变量名 | 类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| PersonSysNo| int| 是 | 人员编码 |
| CardNo| string| 是 | 银行卡号或支付宝账号 |


#### 返回参数

无
